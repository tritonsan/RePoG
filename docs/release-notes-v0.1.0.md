# RePoG v0.1.0 Release Notes Draft

RePoG v0.1.0 is the first public starter release of RePoG as a GM workspace for
agentic coding tools.

RePoG turns a repo folder into a long-form solo RPG campaign notebook. The
agent uses local instructions, Markdown memory, workflows, and small checks to
build worlds, run scenes, track NPCs and secrets, close arcs, and prepare the
next act.

## Highlights

- Codex-first `AGENTS.md` instructions.
- Guided Session 0 campaign creation.
- Markdown campaign memory for world, NPCs, factions, places, secrets,
  relationships, progression, and arc closure.
- Hidden-knowledge boundaries to reduce NPC knowledge leaks.
- Small local helper tools for checking campaign shape, scanning player-facing
  text, and taking snapshots.
- Generic starter examples for fantasy, sci-fi/noir, and historical intrigue.

## Notes

RePoG is not a standalone app or virtual tabletop. It is a workspace pattern for
agentic coding tools, currently best tested with OpenAI Codex-style workflows.
