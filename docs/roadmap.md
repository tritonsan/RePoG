# Roadmap

This roadmap tracks practical improvements for RePoG as an open-source
agentic RPG workspace.

## v0.1.x: Public Usability

- Improve first-run documentation.
- Add more generic starter examples.
- Tighten setup guidance for Codex and other agentic coding tools.
- Collect playtest reports around long-session continuity.
- Improve issue templates and contribution flow.

## v0.2.x: Better Campaign Operations

- Add more audit guidance for hidden knowledge leaks.
- Improve examples of major arc closure, rewards, and companion upgrades.
- Add optional check helpers only when they catch real campaign problems.
- Improve the optional local dashboard after playtesting it in real campaigns.
- Improve visual workflow guidance for tools that can generate or display
  images.

## v0.3.x: Portability

- Add notes for Claude Code, OpenCode, and other agentic coding tools where
  users confirm working workflows.
- Separate Codex-specific tips from general agent workspace conventions.
- Add more sample campaign seeds contributed by users.

## Good First Issue Ideas

- Add a new generic sample campaign seed.
- Improve a confusing section of `START_HERE.md`.
- Add a short transcript showing good GM behavior.
- Report a continuity issue from a long session.
- Improve one template field description.
- Add non-Codex setup notes after testing them.
