# Open Campaign

Open this extracted folder in your agentic coding tool, start a new
conversation, and send:

```text
Start this RePoG campaign and guide me through Session 0.
```

The first setup question should ask only whether you want Quick, Standard, or
Deep Session 0. The active campaign memory is already prepared under
`campaign/`.

Later in System Fit, choose the Turn Protocol after the agent shows its timing,
continuity, dashboard, and image-generation tradeoffs. Fast is recommended but
remains an explicit choice.
