# Open Campaign

Open this extracted folder in your agentic coding tool, start a new
conversation, and send:

```text
Start RePoG and guide me through setup.
```

The first setup question asks only whether you want an **RPG Campaign** or an
**AI Companion**. The second question asks whether you want Quick, Standard,
or Deep Session 0. The active memory is already prepared under `campaign/`.

In RPG setup, later in System Fit, choose the Turn Protocol after the agent shows its timing,
continuity, dashboard, and image-generation tradeoffs. Fast is recommended but
remains an explicit choice. Companion setup instead creates one persistent
fictional character, their independent life, conversation style, memory and
privacy policy, disclosure and deception boundaries, and evidence-based
relationship context. Its separate lightweight Companion View is optional and
off by default; the RPG Dashboard never runs in Companion mode.
