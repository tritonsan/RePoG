# Dashboard Assets

Store player-visible dashboard images here.

Recommended paths:

- `assets/locations/<image-name>.png`
- `assets/characters/<image-name>.png`
- `assets/factions/<image-name>.png`
- `assets/items/<image-name>.png`

Only accepted, player-visible images should be referenced from
`dashboard_state.json`.

Do not place secret GM images here unless they are already safe for the player
to see.
