# World Voices Memory

This is private GM campaign memory. `index.json` owns artifact identity,
lifecycle, threads, version links, private epistemic basis, claim positions,
distribution records, continuity revision, and permanent operation ids.
`artifacts/` owns Markdown bodies.

`knowledge_boundaries.md` remains the only authority for current facts and
holders. Fact ids referenced here do not turn an artifact's claims into world
truth.

The browser never reads this directory. Player-safe, acquired documents are
projected separately below `dashboard/assets/world_voices/`. Keep only active
or pending artifact/thread references in hot campaign preparation and load
this archive on demand.
