# Issues

Campaign id: `new_campaign`

Issues are active pressures, not plot scripts. They tell Codex what is moving
in the world if the player does nothing.

This file owns the durable definition of a systemic problem: what is wrong,
who is affected, and the broad escalation implied by inaction. It does not own
the live scene, the player's dramatic question, a faction's stable identity,
or the latest evaluated offscreen move. Those belong respectively to
`current_state.yaml`, `threads.md`, faction notes, and `world_dynamics.md`.
