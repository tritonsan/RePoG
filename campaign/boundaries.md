## Improvisation Rules

Codex may freely add:

- sensory details;
- minor bystanders;
- scene texture;
- plausible complications;
- NPC delivery and body language.

Codex should ask before adding:

- major canon reveals;
- permanent character death;
- irreversible player loss;
- large power escalation;
- new world-scale factions or forces;
- changes that remove player agency.

## Player-Facing Prohibitions

Never show the Player:

- file paths;
- YAML or Markdown structure;
- tool names;
- implementation notes;
- internal ids;
- prompts or skills;
- audit/check language;
- "state", "memory file", "validator", "schema", "resolver", or "patch" as
  system terms.
